About project

A little look under hood

You maybe interested about how we make forecasts. The heart of our system is located in R programming language. R has been chosen because it is one of the best open source projects for statistical computing. We simply take your data, make some transformations and pass it to one of scripts written for R. Web Server that receives data and communicates with R is written by the Java programming language . All source codes are open and available by GitHub.

Simple scheme of working the service

1.) Java service receives a xls file and an additional information about forecasting settings.

2.) Java service reads the xls file, makes some transformations.

3.) Java service sends the data and the forecasting settings to R server.

4.) R server runs scripts with user's data and makes a forecast.

5.) R server sends that forecast to Java service.

6.) Java service builds a xls with forecast and sends you an email with link to that file.
